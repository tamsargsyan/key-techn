## Setup

To run this project, install it locally using npm:

#### Clone this repository

```bash
git clone <url-to-this-repo>
```

### Install NPM packages

```
cd <project_name>
npm install
```
#### Start working:
```
npm start
```
### Building a distribution version:
```
npm run build
```
